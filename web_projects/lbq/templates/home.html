    As already communicated through various channels by CIO, driven by both financial and performance factors, CIO has initiated a program to transition our Cirrus IBM Cloud infrastructure from the current classic clusters to VPC-based clusters.

    CRAFT has successfully migrated to the VPC-based clusters. Please bookmark the new URL -> New CRAFT URL

    There are several benefits to this transition:

  • Newer and faster hardware
  • Improved network bandwidth (approx. 10x)
  • Firewall flows in seconds instead of hours
  • Better distribution of workloads for less noisy neighbors
  • Application specific domain names and certificates for greatly improved security.
